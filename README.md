# Sistema Web para Clínica / Planos de Saúde

Aplicação web institucional responsiva desenvolvida com HTML, CSS e JavaScript puro, com foco em simulação de planos de saúde e interatividade com o usuário.

## 🚀 Funcionalidades

- Layout responsivo (mobile e desktop)
- Menu de navegação interativo
- Simulação de planos com cálculo dinâmico (IMC e comparação)
- Manipulação do DOM com JavaScript
- Tema escuro (dark mode)
- Estrutura organizada com CSS e variáveis

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📚 Objetivo do projeto

Projeto desenvolvido com fins acadêmicos e de aprendizado, com foco em prática de desenvolvimento front-end e lógica de programação.
